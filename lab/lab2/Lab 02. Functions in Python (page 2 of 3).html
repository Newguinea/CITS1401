The CodeRunner questions you met earlier in this lab asked you to write a program to some specification. However, most of the CodeRunner questions in this course will ask you to write a function to some specification. This simplifies the questions because we don't need to specify exactly what names to use for certain variables, nor exactly how to initialise them or read external data into them.

To get the hang of a CodeRunner function question, here's one to which you should already know the answer.

The question

Write a function to_celsius(fahrenheit) that takes a float temperature in degrees fahrenheit and returns the equivalent temperature in degrees celsius. The formula is:

degrees_celsius = (fahrenheit - 32) * (5 / 9)

See the examples below for expected output.

Note that you must write in only the function implementation itself, without any of the tests or execution of function statements. The first two lines of the answer have been provided in the answer box.

The penalty regime for this question is 0, 10, 20, ..., meaning that you can submit one wrong answer without penalty but thereafter you will pay a penalty of 10% for each wrong submission. So if you get it wrong first time and are not sure why, ask a lab facilitator for help.
